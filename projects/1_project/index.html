GIDEA is an AI-driven simulation platform that streamlines human-computer interaction (HCI) experiments by automating user interactions and experiment management. Traditional HCI studies require extensive manual effort, but GIDEA accelerates this process using large language models (LLMs) to generate realistic interaction scenarios dynamically.

Key Features

  • LLM-Driven Virtual Participants: Simulated avatars with unique personalities engage in real-time interactions based on structured research prompts.
  • Real-Time Experiment Control with Unity: Researchers define objectives, control environments, and monitor experiments dynamically.
  • Virtual Reality (VR) Integration: GIDEA supports immersive experimentation with VR setups, enabling embodied interaction studies.
  • Case Study Validation: The platform has been tested against real-world HCI studies, demonstrating statistically consistent results.

Experiment Workflow

  1. Define Research Questions: Researchers specify objectives and experimental conditions.
  2. Simulate Interactions: LLM-driven avatars engage in realistic interactions.
  3. Monitor & Adjust: The Unity-based dashboard provides real-time control.
  4. Analyze & Compare: Simulated results are validated against real-world data.

With GIDEA, researchers can conduct scalable, efficient, and adaptive HCI studies, accelerating intelligent system evaluation.
